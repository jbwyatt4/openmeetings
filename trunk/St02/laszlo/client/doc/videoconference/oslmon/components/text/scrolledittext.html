scrolledittext._placeTextY

Overrides a parent method which calculates y position for text in a way that doesn't make sense for this class. We just trap the call and do nothing. This field is positioned in y by the constraint y=${parent.border} Note: although this method is a no-op, it still has to be declared, so the parent class's method does not get called on instances of this class.
